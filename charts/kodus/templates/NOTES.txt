{{- $warnAuth := or (not $cfg.NEXTAUTH_URL) (hasPrefix "http://" ($cfg.NEXTAUTH_URL | toString)) }}

⚠️  ATTENTION — review these (they won't block startup, but features depend on them):
{{- if $warnHost }}
   • WEB_HOSTNAME_API is not a public hostname — connecting a Git repo (webhook
     registration) will fail with "Error saving repositories". Set it and
     API_<provider>_CODE_MANAGEMENT_WEBHOOK to a public https URL. For local
     testing, front the webhooks service with a tunnel (see charts/README.md).
{{- end }}
{{- if $warnAuth }}
   • NEXTAUTH_URL is empty or http — sign-in and OAuth callbacks need a public
     https URL in production.
{{- end }}
{{- if $warnLLM }}
   • No LLM key set (API_OPEN_AI_API_KEY) — PR reviews won't run until you add one.
     Anthropic/Claude keys go in the same slot. BYOK is required for self-hosted.
{{- end }}
{{- if $warnMongoOcp }}
   • OpenShift + bundled MongoDB: if kodus-mongodb hits ErrImagePull, your cluster
     likely blocks official Docker Hub images. Override the image
     (--set mongodb.bundled.image=mirror.gcr.io/library/mongo:8) or, for
     production, use mongodb.mode: operator / external.
{{- end }}
{{- end }}

4. Self-hosted using Claude/Anthropic? Put the Anthropic key in API_OPEN_AI_API_KEY
   (kodus-ai reads the single LLM key from that slot and selects the SDK by model id).
